# Celestia

A lightweight internal content creation and publishing system designed to streamline editorial workflows.

## Overview
Celestia was built to replace slow, fragmented content tools by providing a clean, fast interface for creating, editing, and publishing content. The system focuses on speed, usability, and maintainability.

## Features
- Custom content authoring interface
- Fast, lightweight architecture
- Reusable UI components
- Performance-focused rendering
- Modular design for easy extension

## Tech Stack
- JavaScript
- HTML5
- CSS3

## What This Project Demonstrates
- Real-world UI problem solving
- Workflow optimization
- Scalable frontend architecture
- Component-based development thinking

## How to Run
Clone the repo and open `index.html` in a browser, or serve locally with any static server.

## Purpose
This project demonstrates how I approach simplifying complex editorial workflows and improving internal tools through thoughtful UI architecture.
